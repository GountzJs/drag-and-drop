# Drag and Drop library web component

This is a web component library that allows you to create drag and drop interfaces.

## Tech stack

- Lit
- TypeScript
- Pnpm (Package manager)

## Use local

### Install proyect

```bash
pnpm install
```

### Build

```bash
pnpm build
```

## Set global environment

```bash
pnpm link @gountzlibs/drag-and-drop --global
```


## Add in your project

```bash
pnpm link @gountzlibs/drag-and-drop
```
